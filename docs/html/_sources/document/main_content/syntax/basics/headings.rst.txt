===============
Headings
===============

.. note::

    The overline of each of the headings is optional, if included it needs to be the same length as the underline. For  common syntax purposes include the overline like the examples.

*   This is used to create a heading used for parts. (Heading 1 - **3**.1.1.1.1.1)

    .. code-block:: none

        #########
        Heading 1
        #########

*   This is used to create a heading used for chapters. (Heading 2 - 3.\ **1**\ .1.1.1.1)

    .. code-block:: none

        *********
        Heading 2
        *********

*   This is used to create a heading used for sections. (Heading 3 - 3.1.\ **1**\ .1.1.1)

    .. code-block:: none

        =========
        Heading 3
        =========

*   This is used to create a heading used for subsections. (Heading 4 - 3.1.1.\ **1**\ .1.1)

    .. code-block:: none

        ---------
        Heading 4
        ---------

*   This is used to create a heading used for sub-subsections. (Heading 5 - 3.1.1.1.\ **1**\ .1)

    .. code-block:: none

        ^^^^^^^^^
        Heading 5
        ^^^^^^^^^

*   This is used to create a heading used for paragraphs. (Heading 6 - 3.1.1.1.1.\ **1**\ )

    .. code-block:: none

        """"""""""
        Heading 6
        """"""""""
